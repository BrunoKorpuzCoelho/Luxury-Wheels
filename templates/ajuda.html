Disclamer!:

Este website é alimentado por uma base de dados contendo informações sobre todas as viaturas apresentadas. Caso haja algum problema ao carregar ou enviar o projeto final e as viaturas não sejam corretamente guardadas na base de dados, isso pode afetar o funcionamento adequado do website. É fundamental ressaltar que o website depende das viaturas carregadas na base de dados para operar de maneira eficiente. Além disso, ao carregar novas viaturas para teste, é importante considerar que os links para viaturas em destaque e as mais procuradas podem não funcionar corretamente, uma vez que os identificadores (IDs) dos links estão diretamente correlacionados com as viaturas disponíveis na base de dados. Em caso de dúvidas ou questões, por favor, não hesite em entrar em contato para que possamos resolver qualquer problema de forma ágil e eficaz.

Na maioria das páginas do website, apresento um menu e um rodapé com as seguintes funcionalidades:

Menu:

O logotipo da Luxury Wheels funciona como um botão que nos leva sempre de volta à página inicial "Home".

Dispoem de botões para "Veículos", "Serviços e Extras", "Contactos" e "Sobre Nós", que ao serem clicados redirecionam para as páginas correspondentes.

Na área Cliente, os utilizadores podem realizar o Login, Registar um novo cliente e, uma vez autenticados, aceder ao seu perfil. Caso sejam administradores, têm acesso à página para adicionar novas viaturas à base de dados.

Rodapé:

Ofereco botões de acesso às redes sociais. Estes botões direcionam os utilizadores para as páginas iniciais das redes sociais associadas.

Apresento também um link direto para as "Viaturas Mais Procuradas", permitindo aos utilizadores acederem facilmente a esta categoria.

Outras opções no rodapé incluem "Sobre Nós", "Contactos", "Termos e Condições" e "Política de Privacidade", que ao serem clicados redirecionam para as páginas correspondentes.

Página "Home":

Ao clicar no botão "Procurar", os utilizadores serão redirecionados para a página que apresenta a lista completa dos veículos disponíveis.

Na secção de "Veículos em Destaque", cada veículo apresenta um botão que direciona diretamente para a sua página específica.

Esta é a única página em que, quando um administrador ou proprietário está autenticado, é possível adicionar novas viaturas à base de dados.

Página "Cars":

Nesta página, logo abaixo do menu principal, apresenta-se um sistema de filtragem por data. Ao aplicar o filtro de data, apenas os veículos disponíveis para as datas selecionadas serão exibidos.

Logo abaixo do sistema de filtragem, encontra-se um container que permite filtrar os veículos por diversos critérios. Ao selecionar um filtro e clicar em "Filtrar", apenas os veículos que correspondem aos filtros escolhidos serão mostrados.

Cada veículo na base de dados é exibido nesta página, organizados em ordem alfabética da marca, de A a Z. Cada veículo é acompanhado por detalhes relevantes e a primeira foto associada na base de dados. É importante ressaltar que todas as informações são extraídas diretamente da base de dados.

Todos os containers de veículos possuem um botão "Alugar Agora", que ao ser clicado redireciona o utilizador para a página específica do veículo, onde podem ser encontradas mais informações e a possibilidade de realizar a reserva.

Página "Alugar:id":

Nesta página, uma das mais abrangentes, encontramos uma estrutura complexa que proporciona uma experiência detalhada aos utilizadores:

Do lado direito, exibimos a foto principal da viatura, sendo que o JavaScript percorre a base de dados para mostrar até um máximo de 5 fotos adicionais.

No lado esquerdo, apresento todas as características detalhadas da viatura, todas elas carregadas diretamente da base de dados.

Abaixo das fotos, à direita, encontra-se um container destinado à realização de reservas.

Começando pelas datas, é necessário preencher ambas as datas para que o botão "Avançar" seja exibido. Caso o botão não apareça, indica que a viatura está indisponível para alguma das datas selecionadas.

Logo abaixo das datas, são listados os estabelecimentos predefinidos pela Luxury Wheels, onde os utilizadores podem escolher o local para retirada do veículo.

Em seguida, temos opções de pagamento: pagamento no balcão, onde basta clicar em "Avançar"; pagamento com cartão, onde é necessário preencher os dados solicitados; ou pagamento com criptomoeda, seguindo o mesmo procedimento de preenchimento dos dados.

É importante notar que a data de validade do cartão deve ser inserida no formato MM/AA.

Por fim, apresento o valor diário do aluguer, que corresponde ao valor presente na base de dados, e o valor total, que varia de acordo com o número de dias selecionados para o aluguer da viatura.

Página "Recibo: [Número da Reserva]":

Nesta página, os utilizadores têm a oportunidade de revisar e confirmar os detalhes da sua reserva anteriormente preenchidos.

Após verificar os dados, os utilizadores podem clicar no botão "Confirmar Dados" para validar a reserva.

Uma vez confirmados, os dados da reserva são enviados automaticamente para o endereço de e-mail do cliente. O e-mail de confirmação é totalmente estilizado e inclui todos os detalhes relevantes da reserva. Por favor, note que, devido a limitações técnicas, a exibição correta das imagens no e-mail pode variar dependendo do provedor de e-mail utilizado. Até o momento, a formatação do e-mail funciona de forma ideal no Gmail, mas podem ocorrer problemas de exibição em outros provedores de e-mail, especialmente na versão para PC do Hotmail. Isso pode estar relacionado ao uso de imagens armazenadas em nuvem.

Após a submissão dos dados, todas as informações relevantes são gravadas na tabela de gerenciamento, enquanto as datas de início e término da reserva são registradas na tabela do veículo correspondente.

Página "Registar:"

Nesta página efetuamos o registo de um novo cliente, ao criar o novo cliente o email introduzido receberá os dados de login para ficarem guardados
